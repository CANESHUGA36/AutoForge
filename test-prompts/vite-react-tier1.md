# Vite React - Tier 1: 基础功能测试

## 提示词

Build a personal task manager with Vite + React + TypeScript. Features: add/delete tasks, mark complete, filter by status (all/active/completed), localStorage persistence, responsive design with Tailwind CSS.

## 预期技术栈
- **模板**: vite-react-ts
- **复杂度**: 中等（5-8个功能模块）
- **关键测试点**:
  - 项目初始化是否正确（vite + react + ts + tailwind）
  - 状态管理（useState/useEffect + localStorage）
  - 条件渲染过滤（CSS控制显隐，非条件渲染）
  - 响应式布局
  - 构建验证通过

## 验收标准关注点
- F1: 任务CRUD功能
- F2: 过滤功能
- F3: 数据持久化
- D: UI设计质量
